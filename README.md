# Aroma de Campo – Práctica de JavaScript

Sitio web simple desarrollado como práctica de JavaScript, enfocado en la interacción con el usuario mediante alertas y entradas de datos.

## Descripción
El objetivo del proyecto fue aplicar conceptos básicos de JavaScript, como:
- Uso de `alert` y `prompt`
- Interacción con el usuario
- Lógica condicional básica

## Tecnologías
- HTML
- CSS
- JavaScript
